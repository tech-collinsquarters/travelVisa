## üöÄ Day 1: Launch Day

### Before Publishing
- [ ] Enable HTTPS/SSL on hosting
- [ ] Set up 301 redirect (choose www vs non-www)
- [ ] Add Google Analytics tracking code
- [ ] Add Meta Pixel code
- [ ] Create social media images (og-image.jpg, twitter-image.jpg)
- [ ] Add favicon files

### After Publishing (Do These First!)

#### 1. Google Search Console (15 minutes)
1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Click "Add Property"
3. Enter: `travel.eazyviza.com`
4. Verify ownership:
   - Method: HTML tag
   - Copy verification code
   - Add to `<head>` in index.html: `<meta name="google-site-verification" content="PASTE_CODE_HERE" />`
   - Re-publish site
   - Click "Verify" in Search Console

5. Submit Sitemap:
   - Go to "Sitemaps" section
   - Submit: `https://travel.eazyviza.com/sitemap.xml`
   - Wait for processing (24-48 hours)

6. Request Indexing:
   - Go to "URL Inspection" tool
   - Enter: `https://travel.eazyviza.com/`
   - Click "Request Indexing"
   - Google will crawl within 24-48 hours

#### 2. Bing Webmaster Tools (10 minutes)
1. Go to [bing.com/webmasters](https://www.bing.com/webmasters)
2. Import from Google Search Console (easier) OR add manually
3. Submit sitemap: `https://travel.eazyviza.com/sitemap.xml`

#### 3. Google Analytics 4 (10 minutes)
1. Go to [analytics.google.com](https://analytics.google.com)
2. Create new property: "EazyViza Landing Page"
3. Get Measurement ID (starts with G-)
4. Add tracking code to index.html (see README)

#### 4. Google Business Profile (20 minutes)
1. Go to [business.google.com](https://business.google.com)
2. Create listing (if you have physical office)
3. Add website: `travel.eazyviza.com`
4. Add phone, hours, description
5. Upload photos
6. Verify business
